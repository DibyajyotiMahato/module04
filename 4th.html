Ordered lists of html tags

  1. <html> - The root element that wraps all HTML content on a webpage.
  2. <head> - Contains metadata, links to stylesheets, and other resources.
  3. <title> - Defines the title of the document displayed in the browser tab.
  4. <body> - Encloses all the visible content of a webpage.
  5. <h1> to <h6> - Header tags used for defining different levels of headings.
  6. <p> - Represents a paragraph of text.
  7. <a> - Creates hyperlinks to other pages or resources using the href attribute.
  8. <img> - Embeds images using the src attribute.
  9. <ul> - Defines an unordered (bulleted) list.
  10. <ol> - Defines an ordered (numbered) list.
  11. <li> - Represents a list item inside <ul> or <ol>.
  12. <table> - Creates a table structure.
  13. <tr> - Defines a row in a table.
  14. <td> - Defines a cell within a table row.
  15. <th> - Represents a header cell in a table.
  16. <div> - A block-level container for grouping elements.
  17. <span> - An inline container used to style a part of the text.
  18. <form> - Creates an interactive form for user input.
  19. <input> - Defines input fields in a form.
  20. <button> - Represents a clickable button.
  21. <label> - Defines a label for an <input> element.
  22. <select> - Creates a dropdown list.
  23. <option> - Represents an item inside a <select> dropdown.
  24. <textarea> - Defines a multi-line text input field.
  25. <strong> - Displays text in bold for emphasis.
  26. <br> - Inserts a line break.
  27. <hr> - Adds a horizontal rule to separate content.
  28. <link> - Connects external resources like stylesheets to the document.
  29. <script> - Embeds or links JavaScript code in the document.
  30. <style> - Contains internal CSS rules for styling elements.
