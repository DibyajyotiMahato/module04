1. HTML

    primary use case key features/benifits
    Building the structure of web pages.
    • simple and easy to learn.
    • compitable with all web browsers.

  2. CSS

    primary use case key features/benifits
    Styling and layout of web pages.

    • allows for separetion of content and presentation.

  3. JS

    primary use case key features/benifits
    Make web pages more dynamic.
    • client-side execution.
    • platform independence and the ability to create interactive web experiences
    • interoperability with other languages
